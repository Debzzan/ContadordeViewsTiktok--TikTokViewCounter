# 📊 TikTok Views Counter / Contador de Views do TikTok

## 🇺🇸 English

### 📌 Description
A simple tool that counts the **total number of views** from a TikTok user's videos.

---

### ⚙️ Requirements
- Node.js

---

### 🔑 API Key Requirement

You **must have an API key (ID_KEY)** from RapidAPI before running the project.

#### Steps:
1. Go to: https://rapidapi.com/hub  
2. Create a free account  
3. Subscribe to the TikTok Scraper API  
4. Copy your API key  
5. Paste it into the `server.js` file  

API Sign-up
https://rapidapi.com/tikwm-tikwm-default/api/tiktok-scraper7

API endpoint:  
https://rapidapi.com/tikwm-tikwm-default/api/tiktok-scraper7/playground/apiendpoint_7a7e6ee5-c4b7-454d-95a1-96599f5400c3

---

### 🚀 How to Run

⚠️ **Warning:** Make sure you have added your API key in `server.js` before running the project.

1. Open your terminal  
2. Navigate to the folder where `server.js` is located  
3. Run the following command:  
   `node server.js` 

4. To open the interface, run:  
   `open index.html` 

5. This application does not support direct usernames, meaning that you will need to find tiktok user ID. To do this, visit https://fameswap.com/tool-tiktok-user-id and enter the username you would like. 
`Ex.: Nike has @nike as its username, so on this website, type nike to find its TikTok ID, and then enter it in the application's search bar (TikTok ID is often a numeric sequence, such as: 208464585232822272)`
